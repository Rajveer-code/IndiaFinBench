How it works
1
Enter a public HF model ID. It will be loaded via transformers.pipeline with deterministic decoding (do_sample=False).
2
Run a smoke test (10 items) first to confirm the model loads and responds correctly before the full run.
3
Full evaluation scores all 150 items zero-shot. Scoring is task-aware: fuzzy match for REG/TMP, numeric match for NUM, exact Yes/No for CON.
4
Results are saved to the leaderboard immediately. Reload the page after completion to see updated rankings.
Note: Models larger than 30B parameters may time out on CPU-only servers. Run large models locally using python app.py.
Research Benchmark
About IndiaFinBench
The first expert-annotated benchmark for evaluating LLMs on Indian financial regulatory text from SEBI and RBI corpora, spanning 2000–2026.
Task Types
REG
Regulatory Interpretation · n = 53
Extract compliance rules, deadlines, and financial thresholds from SEBI and RBI circulars.
NUM
Numerical Reasoning · n = 32
Arithmetic over repo rates, capital adequacy ratios, and numerical financial thresholds.
CON
Contradiction Detection · n = 30
Identify whether two regulatory passages contradict each other (Yes/No binary).
TMP
Temporal Reasoning · n = 35
Sequence regulatory amendments, effective dates, and supersession events across circulars.
Key Finding

All 9 evaluated LLMs surpass the human expert baseline of 60.0%. Claude 3 Haiku achieves 91.3% — a +31.3 pp margin over human experts. Numerical Reasoning shows the largest inter-model spread (31.3 pp), making it the most discriminating task. Temporal Reasoning accounts for 54% of Claude 3 Haiku's failures. Llama 4 Scout achieves a perfect 100% on Contradiction Detection.
